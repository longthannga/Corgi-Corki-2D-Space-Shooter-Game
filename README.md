## Installation and Usage

1. **Clone the Repository**:
   ```bash
   git clone https://github.com/longthannga/Corgi-Corki-2D-Space-Shooter-Game.git
   cd Corgi-Corki-2D-Space-Shooter-Game
   ```
2. Compile and Run:
  Ensure you have Java installed on your system.
  Compile the Java files:
  ```bash
  javac *.java
  ```
  Run the game:
  ```bash
  java Runner
  ```
